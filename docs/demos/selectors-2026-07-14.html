3. HTML case-insensitive attribute matching

Selectors use UPPERCASE values; elements carry lowercase — they still match, per the HTML spec's attribute list.

p[align=center] — [align="CENTER"]

p[lang=en] — [lang="EN"] (badge added via ::before)

p[dir=ltr] — [dir="LTR"]
